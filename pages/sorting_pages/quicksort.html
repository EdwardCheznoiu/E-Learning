    Aplicatii

    Interactiv

Algoritmul este de tip divide et impera; el sorteaza o secventa a tabloului (initial intreg tabloul), astfel:


  • se alege un element special al listei, numit pivot
  • se ordoneaza elementele listei, astfel incat toate elementele din stanga pivotului sa fie mai mici sau egale cu acesta, si toate elementele din dreapta pivotului sa fie mai mari sau egale cu acesta
  • se continua recursiv cu secventa din stanga pivotului si cu cea din dreapta lui.

    • QuickSort sau Sortarea rapida este o metoda eficienta de sortare a unui tablou, descoperita in 1960 de programatorul britanic C.A.R. Hoare


    Etapa Divide: Vectorul x, format din elementele aflate intre indicii i si j, notat in continuare x[i..j], este impartit in doi subvectori x[i ..m] si x[m+1 ..j], fiecare subvector avand cel putin un element si fiecare element din primul subvector fiind mai mic sau egal decat oricare element din cel de-al doilea subvector.

    Etapa Impera: Cele doua parti sunt sortate apeland recursiv, pana cand se ajunge la subvectori formati dintr-un singur element.

    Combinarea: Avand in vedere ca fiecare subvector este sortat si ca cel mai mare element din subvectorul stang este mai mic decat cel mai mic element din subvectorul drept, atunci intregul vector va fi sortat.


    Complexitate


    Alegerea unui pivot


    Exista patru moduri mai des intalnite pentru alegerea unui pivot

  • Alegerea primului element
  • Algerea ultimului element
  • Alegerea unui element la intamplare
  • Alegerea elementului din mijloc

    • Agloritmul Quicksort


    Algoritmul quicksort este compus din doi algoritmi, unul de partitionare al vectorului folosindu-se de pivot si unul care face sortarea


    Algoritmul de partitionare


  • Se declara doua variabile low si high
  • In low se stocheaza valoarea primului element si in high se stocheaza valoarea ultimului element
  • Se declara o variabila j si se initializeaza cu low
  • Se va lua pivotul ultimul element din vector(in cazul nostru)
  • Se parcurge tot vectorul utilizand variabila i si vom compara fiecare element cu pivotul
  • Daca vec[i] < pivot atunci vom incrementa j si vom face interschimbarea elementelor de la pozitia i si j
  • Cand bucla se incheie se returneaza j - 1

    • Algoritmul de sortare

  • Se va lua pivotul ultimul element din vector(in cazul nostru)
  • Se va gasi indexul corect la pivotului folosindu-se alogritmul de partitionare
  • Se va chema recursiv functia Quicksort cu subvectorul din stanga
  • Se va chema recursiv functia Quicksort cu subvectorul din dreapta



    • O secventa de cod in C++ ar fi urmatoarea
